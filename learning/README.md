# Learning Domain

Schema specification for the **Learning** domain in OPuN networks. This domain covers course catalogs, enrollments, progress tracking, and certificate issuance on Beckn Protocol v2.0.

## Scope

- **Course catalog** — Search and discovery of courses (category, level, type, schedule, fees)
- **Enrollment lifecycle** — Select → Init (enrollment form) → Confirm → Status → Update (progress/certificate) → Cancel
- **Progress and completion** — Progress percentage, content completion, certificate issuance with optional verifiable credentials
